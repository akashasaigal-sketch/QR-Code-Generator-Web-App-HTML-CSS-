📱 QR Code Generator

A simple and user-friendly QR Code Generator built with HTML, CSS, and JavaScript. This web application allows users to enter any text or URL and instantly generate a QR code that can be scanned using any QR code scanner.

🚀 Features
Generate QR codes from text or URLs
Clean and responsive user interface
Instant QR code generation
Lightweight and easy to use
Built with pure HTML, CSS, and JavaScript
🛠 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
QR Code API
📖 How It Works
Enter any text or website URL in the input field.
Click the Generate QR Code button.
The application creates and displays a QR code instantly.
Scan the QR code with any mobile device to access the encoded content.
